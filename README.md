<h1  align="center">TDU History Log</h1>

## is何？

[着座履歴記録アプリ「リレキログ」](https://www.dendai.ac.jp/news/20201116-01.html)のPWAバージョン。

## Usage

依存関係をインストールしてください。

```bash
yarn install --dev
```

### Build and Run

```bash
# build
yarn build

# start
yarn start

# dev start
yarn dev

# create static file
yarn export
```

### Linter

- GTSを使用

```bash
# lint
yarn lint

# clean
yarn clean

# fix
yarn fix
```
